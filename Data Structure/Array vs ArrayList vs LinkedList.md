Array 는 인덱스로 빠르게 검색

ArrayList는 데이터를 찾는데는 빠르지만 수정 삭제가 느림

LinkedList는 데이터 삽입과 삭제가 빠름

## Array vs List

Array는 사이즈를 정해야줘야함

List는 동적으로 사이즈를 늘릴 수 있음

### LinkedList vs ArrayList

순차적으로 추가/삭제하는 경우에는 ArrayList가 LinkedList보다 빠름

중간 데이터를 추가/삭제하는 경우에는 LinkedList가 ArrayList보다 빠름
